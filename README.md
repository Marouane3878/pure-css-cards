# CSS Playing Cards
[Live Demo] (https://marouane3878.github.io/pure-css-cards/)
A simple playing cards UI built using pure HTML and CSS.  
This project demonstrates Flexbox layout, basic styling, and hover interactions without any JavaScript.

## Features
- Responsive card layout using Flexbox
- Classic playing card design
- Hover and active card effects
- Pure HTML & CSS

## Technologies
- HTML5
- CSS3

## Preview
Open `index.html` in your browser to view the cards.

## Purpose
This project was created for learning and practicing frontend layout and styling techniques.
